This project is mini-demo "Notebook" Api.

# 1. Features

- Freamwork: Moleculer
- Api Gateway: Moleculer-web
- ORM: Sequelize
- Validator: Joi

---

# 2. Usage

- Clone project
- Run:

```
npm i
```

- Create .env file follow .env-simple
- Create database and fill .env
- Start project:

```

node ./src
```
